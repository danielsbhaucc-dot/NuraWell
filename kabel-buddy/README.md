# כבל-באדי ⚡

מחשבון ידידותי לבחירת **קוטר כבל** לפי **תקנות החשמל (העמסה והגנה על מוליכים מבודדים וכבלים במתח נמוך), תשע"ד-2014**.

## מה זה עושה?

שאלון פשוט בעברית שמחליף את הטבלאות המסובכות:
- שיטת התקנה (תוספת שלישית)
- סוג בידוד (70°C / 90°C)
- חד/תלת פאזי, נחושת/אלומיניום
- זרם עומס (ישיר או מחישוב הספק)
- תיקוני טמפרטורה וקיבוץ כבלים (תוספת ראשונה)

## איך פותחים?

פתחו את `index.html` בדפדפן — אין צורך בהתקנה.

```bash
# אופציונלי — שרת מקומי
npx serve .
```

## מקור הנתונים

- [תקנות החשמל בנבו](https://www.nevo.co.il/law_html/law00/131976.htm) (עדכון: 17.02.2026)
- טבלאות Iz: תוספת שנייה
- מקדמי תיקון: תוספת ראשונה (טבלאות 1–7)
- שיטות התקנה: תוספת שלישית

## חשוב

הכלי מסייע בחישוב לפי התקן — **האחריות המקצועית נשארת אצל החשמלאי המוסמך**.

בנוי עבור אבא שלך וכל החשמלאים שרוצים שאלון נוח במקום לדפדף בטבלאות 📋
